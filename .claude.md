## Development Guidelines

1. **Testing First**: 모든 변경사항은 테스트 통과 확인 후 완료 처리
2. **Type Safety**: `@catchvoca/types`에서 타입 import 사용
3. **Repository Pattern**: Dexie 직접 호출 금지, Repository 사용 필수
4. **Build Verification**: Extension 수정 시 반드시 `pnpm build:extension` 실행
